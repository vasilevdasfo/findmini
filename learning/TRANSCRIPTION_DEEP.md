## 🏗 4 канала транскрибации — архитектура Димы

Дима не делает «один бот всё умеет». У него 4 отдельных входа:

### 1. `@DEMIIDEAS_bot` — форварды из других чатов
Ты форвардишь голосовое от клиента → бот транскрибит → кладёт в `INBOX.md` с пометкой «от такого-то».
**Назначение:** архив переписок, чтобы не слушать заново.

### 2. `@vdavc_bot` — персональные диктовки
Только для своего голоса. Быстрый старт, короткие заметки.
**Назначение:** voice-daily/`YYYY-MM-DD.md` — дневник мыслей.

### 3. Obsidian + Wispr Flow
Диктуешь прямо в Obsidian в файл `VOICE_INBOX.md`. Без бота, напрямую в vault.
**Назначение:** длинные размышления, когда сидишь за маком.

### 4. Чат-транскрибация в Claude Code
В открытом окне Claude Code скидываешь голосовое (через Telegram Desktop drag-drop) → Claude транскрибит и отвечает сразу.
**Назначение:** диалог голосом, итерация над идеей.

**Почему 4 а не 1:** разные контексты = разные целевые файлы. Один бот «на всё» превращается в свалку.

## 🔧 Как использовать Groq правильно (русский язык)

### Минимальный код
```python
from groq import Groq
client = Groq(api_key=GROQ_KEY)

with open("voice.ogg", "rb") as f:
    result = client.audio.transcriptions.create(
        file=("voice.ogg", f.read()),
        model="whisper-large-v3",
        language="ru",                    # ← обязательно!
        response_format="verbose_json",   # ← даст timestamps
        temperature=0.0,                  # ← без фантазий
        prompt="Дмитрий, Александр, WEX, vda.vc"  # ← словарь имён!
    )
print(result.text)
```

### 4 параметра которые все забывают

1. **`language="ru"`** — без него Whisper угадывает (и иногда думает что ты говоришь по-украински).
2. **`temperature=0.0`** — дефолт 0.2, иногда «додумывает» слова. Ставь 0.
3. **`prompt`** — словарь твоих имён, терминов, брендов. Whisper ищет их приоритетно. Без промпта «Криптокомплаенс» превратится в «крипт комплаенс». С промптом — узнаёт.
4. **`response_format="verbose_json"`** — получишь timestamps по словам. Пригодится для монтажа видео/аудио.

## 🎯 Deepgram для встреч с диаризацией

Установка:
```bash
pip3 install deepgram-sdk
```

Код:
```python
from deepgram import DeepgramClient, PrerecordedOptions

dg = DeepgramClient(DEEPGRAM_KEY)
options = PrerecordedOptions(
    model="nova-2",
    language="ru",
    diarize=True,              # ← кто говорит
    punctuate=True,
    smart_format=True,
    utterances=True,           # ← разбивка по репликам
)

with open("meeting.m4a", "rb") as f:
    response = dg.listen.prerecorded.v("1").transcribe_file(
        {"buffer": f.read()}, options
    )

# Результат с разметкой speaker 0 / speaker 1
for utt in response.results.utterances:
    print(f"[Speaker {utt.speaker}]: {utt.transcript}")
```

Результат:
```
[Speaker 0]: Александр, давай обсудим сроки.
[Speaker 1]: Готов, у меня вопрос по бюджету...
[Speaker 0]: Слушаю.
```

**Цена:** 60-минутная встреча = 25 центов. За 100 встреч — $25/мес.

## 🎙 Wispr Flow — диктовка в любое приложение

Это не API, а десктопное приложение. Работает поверх macOS.

1. Скачай с https://wisprflow.ai → установи
2. `$15/мес` подписка
3. Hotkey (по умолчанию Fn) — зажал, говоришь, отпустил → текст вставляется в активное поле

**Когда использовать:**
- Пишешь в Telegram/Slack/email → жмёшь Fn, говоришь → готово.
- В Claude Code → диктуешь промпт вместо печати.
- В Obsidian → диктуешь заметку прямо в открытый файл.

**Почему Дима держит и Wispr, и свой бот:**
- Wispr — для «здесь и сейчас», 1 фраза.
- Бот — для длинного голосового «накидал мыслей на 3 минуты, разбери».

## ⚠️ Грабли транскрибации (набиты кровью)

1. **ogg Opus → Whisper не ест напрямую.** Конвертируй через ffmpeg в wav 16kHz mono:
   ```bash
   ffmpeg -i voice.ogg -ar 16000 -ac 1 voice.wav
   ```
2. **Голосовые > 25 МБ Groq не берёт.** Режь на куски:
   ```bash
   ffmpeg -i long.mp3 -f segment -segment_time 600 -c copy out_%03d.mp3
   ```
3. **Whisper галлюцинирует на тишине.** Если 5+ секунд молчания → вернёт «Продолжение следует…» или «Субтитры сделал DimaTorzok». Режь тишину через `ffmpeg silencedetect` заранее.
4. **Имена пишет как слышит.** «Бычковский» → «бычковский», «Васильев» → «васильев». Не полагайся на транскрипт для поиска — делай нормализацию через Claude.
5. **Не храни сырое аудио вечно.** После транскрипта удаляй `.ogg` — иначе ~/Library забьётся за месяц.
